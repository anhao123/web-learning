###Pull request 
    请大牛帮忙改bug 但又不能将其邀请为自己团队的成员
      大牛 就需要 fork 代码
          1、 找到仓库 fork  就可以将此仓库复制到自己仓库 
          2、 本地 clone 下载  改bug  留言改完
          3、需求方 可以 在GitHub上 合并改好的代码
          
      如果几天之后 还需要请大牛帮忙
         大牛 两种做法 
            一 、 删了之前 fork 的仓库   再次fork一下
            二 、 在之前fork 的库中  再配一个别名 地址是请求方的仓库地址    git fetch 别名 更新数据
