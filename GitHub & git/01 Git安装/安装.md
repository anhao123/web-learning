### linux命令使用Git

    $ git --version  查看git版本信息
    $ git config --list  查看配置信息
    $ git config --global user.name "an" 用户名
    $ git config --global user.email "an" 邮箱
       配置信息中的
       --sysytem  操作系统中的配置只用于所有用户
      --global    用户目录下的配置只适用于该用户
        不写       此配置只适用于 该项目
