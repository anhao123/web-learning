### 1、初始化一个项目
   *新建一个文件 
   *$ git init  初始化
     
    hooks    目录包含客户端或服务端的钩子脚本;
    info     包含- -个全局性排除文件
    logs       保存日志信息
    **objects**        目录存储所有数据内容;
    **refs**        目录存储指向数据(分支)的提交对象的指针

     **HEAD**       文件指示目前被检出的分支
     **index**       文件保存暂存区信息
